    Create a `"use client"` component with this 4-state machine:
    - `idle` — textarea input + optional name field + distill button
    - `distilling` — spinner ("Distilling grammar…"), button disabled
    - `done` — blueprint viewer + "New distillation" reset button
    - `error` — error message + "Try again" button (resets to idle, preserves textarea content)

    **Props:** `{ projectId: string }`

    **Prompt parsing:**
    Split textarea value by `\n`, filter out empty/whitespace-only lines:
    ```typescript
    const prompts = text
      .split("\n")
      .map((l) => l.trim())
      .filter((l) => l.length > 0)
    ```
    Show live count: `"N prompt{N !== 1 ? 's' : ''} detected"` — update as user types.

    **Optional name field:**
    Small `<Input>` below the textarea, placeholder "Blueprint name (optional)".
    Passed as `name` in the API call when non-empty.

    **State shape:**
    ```typescript
    type Phase = "idle" | "distilling" | "done" | "error"
    // Track: phase, text (textarea value), name (input value),
    //        parsedBlueprint (ParsedBlueprint | null), errorMsg (string)
    ```

    **ParsedBlueprint interface and helper (co-located in this file):**
    ```typescript
    interface ParsedBlueprint {
      sequence_pattern: GrammarBlueprintExtraction["sequence_pattern"] | null
      density: "sparse" | "medium" | "dense" | null
      avg_length: number | null
      compression_style: string | null
      sentence_structure: string | null
      qualifier_placement: string | null
      characteristic_phrases: string[]
      style_vocabulary: string[]
      summary: string | null
      raw_prompts: string[]
    }

    function parseDbBlueprint(raw: GrammarBlueprint): ParsedBlueprint {
      const distilled = raw.distilled_grammar
        ? (JSON.parse(raw.distilled_grammar) as GrammarBlueprintExtraction)
        : null
      return {
        sequence_pattern: raw.sequence_pattern
          ? (JSON.parse(raw.sequence_pattern) as GrammarBlueprintExtraction["sequence_pattern"])
          : null,
        density: raw.density as "sparse" | "medium" | "dense" | null,
        avg_length: raw.avg_length,
        compression_style: raw.compression_style,
        sentence_structure: distilled?.sentence_structure ?? null,
        qualifier_placement: distilled?.qualifier_placement ?? null,
        characteristic_phrases: distilled?.characteristic_phrases ?? [],
        style_vocabulary: distilled?.style_vocabulary ?? [],
        summary: distilled?.summary ?? null,
        raw_prompts: raw.raw_prompts
          ? (JSON.parse(raw.raw_prompts) as string[])
          : [],
      }
    }
    ```

    **Distill function:**
    ```typescript
    async function distill() {
      const promptList = text.split("\n").map((l) => l.trim()).filter((l) => l.length > 0)
      setPhase("distilling")
      const res = await fetch("/api/distill", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          projectId,
          prompts: promptList,
          ...(name.trim() ? { name: name.trim() } : {}),
        }),
      })
      if (!res.ok) {
        const data = await res.json() as { error?: string }
        setErrorMsg(data.error ?? "Distillation failed")
        setPhase("error")
        return
      }
      const data = await res.json() as { id: string; blueprint: GrammarBlueprint }
      setParsedBlueprint(parseDbBlueprint(data.blueprint))
      setPhase("done")
    }
    ```

    **Blueprint viewer sections (done state):**

    1. **Grammar Summary** — full-width prominent text block at top
    2. **Sequence Pattern** — Card with two rows:
       - Template: monospace code block showing the template string
       - Elements: element tags rendered as `<Badge variant="outline">` pills
    3. **Writing Mechanics** — Card with field rows:
       - Density: `<Badge>` colored by value (sparse=secondary, medium=default, dense=destructive)
       - Avg length: `{avg_length} chars`
       - Compression: text value
       - Sentence structure: text value
       - Qualifier placement: text value
    4. **Characteristic Phrases** — Card with ordered list of up to 5 items (use monospace/code styling)
    5. **Style Vocabulary** — Card with `<Badge variant="secondary">` pill cloud (up to 10 terms)

    Use existing shadcn components: `Card`, `CardHeader`, `CardContent`, `CardTitle`, `Badge`, `Button`, `Input`, `Textarea`, `Separator`.

    **Idle state layout:**
    ```
    [Label: "Reference prompts"]
    [Textarea — rows=10, placeholder "Paste reference prompts here, one per line…"]
    [N prompts detected]    [Label: "Blueprint name (optional)"]
    [Input — placeholder "e.g. Midjourney cinematic style"]
    [Distill Grammar button — disabled when 0 prompts]
    ```

    **Avoid:**
    - Clearing textarea on error — preserve content so user can retry without re-pasting
    - Hardcoding prompt count in button label — show count in a separate line below textarea
    - Importing AI SDK or server-only modules — `"use client"` only
