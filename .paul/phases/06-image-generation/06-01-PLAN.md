# Plan 06-01: Image Generation Backend

## Goal

Build the server-side image generation pipeline. After this plan:
- `/api/generate` accepts an `outputId`, reads the final prompt, calls the active image gen provider, saves the result, and returns `{ id, url, provider }`
- Nano Banana 2 (`imagen-3.0-generate-002`) via `experimental_generateImage` — base64 → saved to `uploads/`
- Replicate (`black-forest-labs/flux-schnell`) via REST `fetch` with `Prefer: wait` — URL stored directly
- `/api/uploads/[filename]` serves files from the `uploads/` directory
- `reference_image` stored in `design_schemas` so Phase 6 UI can display the original reference image for comparison

## Scope

6 files:
- `src/lib/schema.sql` — add `reference_image TEXT` column
- `src/lib/db.ts` — `ALTER TABLE` migration for existing DBs
- `src/types/db.ts` — `DesignSchema.reference_image: string | null`
- `src/app/api/analyze/route.ts` — store `filename` as `reference_image`
- `src/app/api/uploads/[filename]/route.ts` — new: serve `uploads/` directory
- `src/app/api/generate/route.ts` — new: image generation endpoint

### Task 5 — Create `/api/uploads/[filename]/route.ts`

**File:** `src/app/api/uploads/[filename]/route.ts` (new)

Serve files from the `uploads/` directory. Returns the file with correct `Content-Type` based on extension; 404 if not found.

```typescript
import { NextResponse } from "next/server"
import fs from "fs"
import path from "path"

const MIME: Record<string, string> = {
  ".png":  "image/png",
  ".jpg":  "image/jpeg",
  ".jpeg": "image/jpeg",
  ".webp": "image/webp",
  ".gif":  "image/gif",
}

export async function GET(
  _request: Request,
  { params }: { params: Promise<{ filename: string }> }
) {
  const { filename } = await params
  const filePath = path.join(process.cwd(), "uploads", filename)

  if (!fs.existsSync(filePath)) {
    return NextResponse.json({ error: "Not found" }, { status: 404 })
  }

  const ext = path.extname(filename).toLowerCase()
  const contentType = MIME[ext] ?? "application/octet-stream"
  const buffer = fs.readFileSync(filePath)

  return new Response(buffer, {
    headers: { "Content-Type": contentType },
  })
}
```

**Verify:** `GET /api/uploads/{existing-filename}` returns image bytes with correct `Content-Type`.

### Task 6 — Create `/api/generate/route.ts`

**File:** `src/app/api/generate/route.ts` (new)

POST endpoint. Request body: `{ outputId: string }`.

Logic:
1. Validate request — return 400 if `outputId` missing
2. Load `prompt_outputs` row — return 404 if not found
3. Guard `final_prompt` not null — return 400 if null
4. Call image gen provider via `getImageGenProvider()`
5. Save `generated_images` row — status `"complete"`
6. Return `{ id, url, provider }`

**Nano Banana 2 path** — uses `experimental_generateImage` from `ai` package:
```typescript
import { experimental_generateImage } from "ai"
import { createGoogleGenerativeAI } from "@ai-sdk/google"

const google = createGoogleGenerativeAI({ apiKey: env.GOOGLE_GENERATIVE_AI_API_KEY })
const imageModel = google.image("imagen-3.0-generate-002")

const { image } = await experimental_generateImage({
  model: imageModel,
  prompt: finalPrompt,
})

// image.base64 → save PNG to uploads/gen-{uuid}.png
const genFilename = `gen-${crypto.randomUUID()}.png`
const genPath = path.join(process.cwd(), "uploads", genFilename)
fs.writeFileSync(genPath, Buffer.from(image.base64, "base64"))
const url = `/api/uploads/${genFilename}`
const model = "imagen-3.0-generate-002"
```

**Replicate path** — REST API via `fetch` with `Prefer: wait`:
```typescript
const replicateRes = await fetch("https://api.replicate.com/v1/predictions", {
  method: "POST",
  headers: {
    "Authorization": `Bearer ${env.REPLICATE_API_TOKEN}`,
    "Content-Type": "application/json",
    "Prefer": "wait",
  },
  body: JSON.stringify({
    version: "black-forest-labs/flux-schnell",
    input: { prompt: finalPrompt },
  }),
})

interface ReplicateResponse {
  id: string
  status: string
  output: string[] | null
  error: string | null
}

const replicateData = (await replicateRes.json()) as ReplicateResponse
if (!replicateRes.ok || replicateData.status !== "succeeded" || !replicateData.output) {
  throw new Error(replicateData.error ?? "Replicate generation failed")
}

const firstOutput = replicateData.output[0]
if (!firstOutput) throw new Error("Replicate returned empty output")

const url = firstOutput
const model = "flux-schnell"
```

const imageId = crypto.randomUUID()
db.prepare(

   VALUES (?, ?, ?, ?, ?, 'complete')`

**Full route structure:**
```typescript
import { NextResponse } from "next/server"
import { z } from "zod"
import { experimental_generateImage } from "ai"
import { createGoogleGenerativeAI } from "@ai-sdk/google"
import fs from "fs"
import path from "path"
import crypto from "crypto"
import { getDb } from "@/lib/db"
import { getImageGenProvider, env } from "@/lib/env"
import type { PromptOutput } from "@/types/db"

const RequestSchema = z.object({
  outputId: z.string().min(1),
})

interface ReplicateResponse {
  id: string
  status: string
  output: string[] | null
  error: string | null
}

export async function POST(request: Request) {
  try {
    const body: unknown = await request.json()
    const parsed = RequestSchema.safeParse(body)
    if (!parsed.success) {
      return NextResponse.json({ error: "outputId is required" }, { status: 400 })
    }

    const { outputId } = parsed.data
    const db = getDb()

    const output = db
      .prepare("SELECT * FROM prompt_outputs WHERE id = ?")
      .get(outputId) as PromptOutput | undefined
    if (!output) {
      return NextResponse.json({ error: "Prompt output not found" }, { status: 404 })
    }
    if (!output.final_prompt) {
      return NextResponse.json({ error: "Prompt output has no final_prompt" }, { status: 400 })
    }

    const finalPrompt = output.final_prompt
    const provider = getImageGenProvider()

    let url: string
    let model: string

    if (provider === "nano_banana_2") {
      if (!env.GOOGLE_GENERATIVE_AI_API_KEY) {
        throw new Error("GOOGLE_GENERATIVE_AI_API_KEY is required for Nano Banana 2")
      }
      const google = createGoogleGenerativeAI({ apiKey: env.GOOGLE_GENERATIVE_AI_API_KEY })
      const imageModel = google.image("imagen-3.0-generate-002")
      const { image } = await experimental_generateImage({ model: imageModel, prompt: finalPrompt })
      const genFilename = `gen-${crypto.randomUUID()}.png`
      const genPath = path.join(process.cwd(), "uploads", genFilename)
      fs.mkdirSync(path.dirname(genPath), { recursive: true })
      fs.writeFileSync(genPath, Buffer.from(image.base64, "base64"))
      url = `/api/uploads/${genFilename}`
      model = "imagen-3.0-generate-002"
    } else {
      if (!env.REPLICATE_API_TOKEN) {
        throw new Error("REPLICATE_API_TOKEN is required for Replicate")
      }
      const replicateRes = await fetch("https://api.replicate.com/v1/predictions", {
        method: "POST",
        headers: {
          Authorization: `Bearer ${env.REPLICATE_API_TOKEN}`,
          "Content-Type": "application/json",
          Prefer: "wait",
        },
        body: JSON.stringify({
          version: "black-forest-labs/flux-schnell",
          input: { prompt: finalPrompt },
        }),
      })
      const replicateData = (await replicateRes.json()) as ReplicateResponse
      if (!replicateRes.ok || replicateData.status !== "succeeded" || !replicateData.output) {
        throw new Error(replicateData.error ?? "Replicate generation failed")
      }
      const firstOutput = replicateData.output[0]
      if (!firstOutput) throw new Error("Replicate returned empty output")
      url = firstOutput
      model = "flux-schnell"
    }

    const imageId = crypto.randomUUID()
    db.prepare(
      `INSERT INTO generated_images (id, prompt_output_id, provider, model, url, status)
       VALUES (?, ?, ?, ?, ?, 'complete')`
    ).run(imageId, outputId, provider, model, url)

    return NextResponse.json({ id: imageId, url, provider })
  } catch (error) {
    console.error("[POST /api/generate]", error)
    return NextResponse.json({ error: "Image generation failed" }, { status: 500 })
  }
}
```

**Verify:** Endpoint returns `{ id, url, provider }` with the correct shape.

## Decision Log

| Decision | Rationale |
|----------|-----------|
| `ALTER TABLE` try/catch after `exec(schema)` | `CREATE TABLE IF NOT EXISTS` won't add new columns; `ALTER TABLE` throws on existing column — try/catch is the idempotent pattern for SQLite column migrations |
| `experimental_generateImage` from `ai` package | `@ai-sdk/google` v3 exposes image model via `.image()` method; `ai` package exposes `experimental_generateImage` — no new package needed |
| Save Nano Banana 2 image as PNG to `uploads/` | Base64 response must be materialised; `uploads/` is already the file store; served via new `/api/uploads/[filename]` route |
| Replicate URL stored directly (not downloaded) | External URL is sufficient for local MVP; download-and-save deferred to Phase 7 if needed |
| `Prefer: wait` header for Replicate | Synchronous response — no polling loop, no webhooks, simpler code; works within 60s for flux-schnell |
| `flux-schnell` version identifier | Flux Schnell is the fast model on Replicate; no pinned SHA needed for MVP |
